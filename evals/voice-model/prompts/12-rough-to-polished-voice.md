# Rough To Polished Voice

## Category

Writing

## Writing Mode

Medium-form.

## Input Context

Use a bounded rough excerpt from Kalen's approved local-only writing corpus.

## Task

Turn the source into a polished essay-style passage that sounds like the same writer on a careful day.

## Output Requirements

- 600-1,200 words.
- Preserve the source's controlling idea and practical stakes.
- Keep the writing plainspoken and specific.
- Make paragraph flow stronger without making the prose slick or corporate.
- Use only source-grounded material.
- Add a short title.

## Scoring Notes

Focus on whether the result feels like an improved version of the source rather than an assistant's replacement voice.

## Expected Failure Modes

- Becoming too literary, too executive, or too therapeutic.
- Inventing structure that changes the meaning.
- Removing unresolved questions that should remain alive.
- Writing in generic AI cadence.
