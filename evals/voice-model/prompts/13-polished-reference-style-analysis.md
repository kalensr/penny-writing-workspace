# Polished Reference Style Analysis

## Category

Knowledge Synthesis

## Writing Mode

Medium-form.

## Input Context

Use extracted text from one or more approved local-only polished writing samples.

## Task

Analyze the writing style in the source and produce a compact style brief that could guide future revisions.

## Output Requirements

- 500-900 words.
- Identify observable style traits with evidence types, not long quotations.
- Distinguish voice, structure, sentence habits, argument habits, and revision risks.
- Include a short "use this when revising" checklist.
- Do not quote more than short phrases.
- Do not infer private biography or facts beyond the source text.

## Scoring Notes

Focus on source fidelity, useful style guidance, specificity, and restraint.

## Expected Failure Modes

- Generic style advice that could apply to anyone.
- Overstating conclusions from a narrow sample.
- Quoting too much source text.
- Treating polished samples as a fixed formula instead of evidence.
